Install Git

Install Jenkins
1. Set Password and Enable Authorization
2. Install Git Plugin and MSBuild Plugin

===Setup Git===
1. Add Credentials for Git
2. Specify git.exe path in Manage Jenkins -> "Configure System" or "Global Tool Configuration"
	Name					: Default
	Path to Git executable 	: C:\Users\VEST\AppData\Local\GitHub\PortableGit_25d850739bc178b2eb13c3e2a9faafea2f9143c0\cmd\git.exe
3. Specify git branch in job



===Setup MSBuild===
1. add MSBuild path to "Manage Jenkins -> Configure System" in "Environment variables"
	Name	: MSBuild v4.0.30319
	Value	: C:\Windows\Microsoft.NET\Framework\v4.0.30319\
	
2. add MSBuild path to "MSBuild" section
	Name			: v4.0.30319
	Path to MSBuild : C:\Windows\Microsoft.NET\Framework\v4.0.30319\MSBuild.exe
	
3. in jenkins job select MSBuild Version
	MSBuild Version : v4.0.30319
	
	
===---Publish The App---===
Step 1:
=== Jenkins Setup ===
Add Build Step: Execute Windows batch command
Command 1 (Create ZIP):
"C:\Windows\Microsoft.NET\Framework\v4.0.30319\msbuild.exe" "SampleWeb\SampleWeb.csproj" /T:Build;Package /p:Configuration=DEBUG /p:OutputPath="obj\DEBUG" /p:DeployIisAppPath="/Default Web Site/SampleWeb" /p:VisualStudioVersion=11.0
Command 2 (Publish to IIS):
"C:\Program Files (x86)\IIS\Microsoft Web Deploy V3\msdeploy.exe" -verb:sync -source:package="SampleWeb\obj\Debug\_PublishedWebsites\SampleWeb_Package\SampleWeb.zip" -dest:auto,computerName=VEST-PC -allowUntrusted=true
